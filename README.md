# mac-setup-things
Basically a series of instructions and commands when setting up my basic dev environment.

Basic Developer Software:
1) Install Xcode
`xcode-select --install`

2) Set up Homebrew
`echo export "PATH=\"/usr/local/bin:/usr/local/sbin:~/bin:$PATH\"" > ~/.zshrc`

3) Reload the zshrc
`source ~/.zshrc`
